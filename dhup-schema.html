The DHUP SQL Schema

The DHUP server makes use of one new table, owner:

  • id INT NOT NULL PRIMARY KEY
  • password TEXT
It also extends the domain table as described in the pgsqldns SQL Schema to contain the following:
  • id INT NOT NULL PRIMARY KEY
  • name TEXT NOT NULL
  • owner INT NOT NULL REFERENCES owner(id)
  • last_sequence INT

These tables may contain any additional fields. It is recommended to index on domain(name) for maximum performance.
